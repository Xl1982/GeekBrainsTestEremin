на должность Преподаватель курса «Сбор и разметка данных».

Задание. Разработать и реализовать скрипт на языке Python
с использованием BeautifulSoup для извлечения определенной информации.

Данный скрипт:
1. Обходит заданный сайт, извлекает из текста ссылки по заданному образцу.
2. После извлечения всех ссылок на данной странице, переходит к следующей странице.
3. После прохождения всех страниц, записывает результат в файл.
4. Дублирует результат и время записи в базу данных.

Результаты можно использовать в практических целях:
Для рассылки рекламных сообщений по данным чатам,
через библиотеки телеграм, Telethon, Pyrogram и тп.
